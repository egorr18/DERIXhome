# DERIX Home

DERIX Home — веб-сайт сервісу з підбору, оренди та продажу нерухомості з інтегрованим чат-ботом консультаційної підтримки.

# Опис проєкту
Проєкт розроблено з метою створення інформаційного веб-ресурсу у сфері нерухомості
з інтегрованим чат-ботом, який допомагає користувачам знаходити житло,
отримувати консультації та швидко зв’язуватися з агентом.

Веб-сайт виконує роль інформаційної платформи, а ключовим елементом взаємодії
з користувачем є чат-бот **@DERIXhomebot**, який автоматизує первинну консультацію
та навігацію по сервісу.

# Призначення чат-бота @DERIXhomebot
Чат-бот **@DERIXhomebot** створений для автоматизації комунікації між користувачем
та сервісом нерухомості DERIX Home.

Він виконує роль першої лінії підтримки та помічника у сфері пошуку житла,
дозволяючи користувачам швидко отримати необхідну інформацію
без участі оператора.

Основна мета чат-бота:
- спрощення процесу пошуку житла;
- швидке надання консультацій;
- зменшення навантаження на агентів з нерухомості;
- підвищення зручності взаємодії з веб-сайтом.

# Функціональні можливості чат-бота
Чат-бот **@DERIXhomebot** забезпечує:

- підбір квартир і будинків для оренди або купівлі;
- консультації щодо районів, типів житла та бюджету;
- допомогу зі здачею або продажем нерухомості;
- надання контактної інформації та можливість зв’язку з агентом;
- відповіді на поширені запитання користувачів;
- підтримку контексту діалогу в межах однієї сесії;
- цілодобову роботу (24/7).

Чат-бот реалізований на платформі **Telegram**, що забезпечує стабільну роботу,
зручний інтерфейс та швидку взаємодію з користувачами.

# Інтеграція чат-бота з веб-сайтом
Інтеграція чат-бота **@DERIXhomebot** здійснюється через інтерактивну кнопку,
розміщену на всіх основних сторінках веб-сайту.

Після натискання кнопки користувач автоматично переходить
до діалогу з чат-ботом у середовищі Telegram.

Таке рішення дозволяє:
- використовувати надійну зовнішню платформу для діалогів;
- зберігати контекст спілкування з користувачем;
- забезпечити масштабованість та простоту підтримки системи.

# Функціональні можливості сайту
- Головна сторінка з описом сервісу нерухомості
- Сторінка FAQ з поширеними запитаннями щодо пошуку житла
- Блог з інформаційними матеріалами про нерухомість
- Сторінка контактів з формою зворотного зв’язку
- Інтеграція чат-бота для швидкої консультації

# Аналітика
У проєкті реалізовано модуль клієнтської аналітики, який:
- відстежує кількість переглядів сторінок сайту;
- підраховує кількість взаємодій користувачів з чат-ботом;
- зберігає повідомлення, надіслані через форму зворотного зв’язку.

Дані зберігаються локально у браузері користувача
та можуть бути використані для аналізу поведінки користувачів
і подальшого покращення сервісу.

# Технології
- HTML5
- CSS3
- JavaScript
- Telegram Bot API
- LocalStorage (збереження аналітичних даних)

## Архітектура проєкту
- Клієнтська частина реалізована за допомогою HTML, CSS та JavaScript
- Чат-бот функціонує як окремий сервіс у середовищі Telegram
- Інтеграція чат-бота здійснюється через JavaScript
- Аналітика реалізована без використання серверної частини

# Мета проєкту
Автоматизація процесу пошуку житла,
покращення взаємодії користувачів з веб-сайтом
та зменшення навантаження на агентів з нерухомості.
